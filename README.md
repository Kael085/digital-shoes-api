# API Simples - String do Servidor

## Descrição
Essa é uma API básica que retorna uma string quando acessada. Serve para demonstrar conhecimentos em backend e manipulação de rotas.

## Como rodar
1. Abra o Replit e clique em **Run**.
2. A API vai iniciar e mostrar a URL do servidor.
3. Acesse a URL no navegador ou via Postman para ver a resposta.

## Funcionalidades
- Retorna uma mensagem simples (string) do servidor.
- Demonstra configuração de servidor e rota HTTP básica.

## Aprendizados
- Configuração de um servidor Node.js simples.
- Criação e manipulação de rotas HTTP.
- Teste de endpoints via navegador ou ferramentas externas.
